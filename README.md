# 📊 HR Analytics Dashboard (Tableau)
Project Summary  
   Interactive HR Analytics Dashboard built in Tableau to analyze employee attrition and workforce trends.
Business Objective  
   Identify key drivers of employee turnover and improve retention strategies.
Dataset  
   HR dataset containing employee demographics, salary, department, role, and attrition status.
Key Metrics Analyzed  
   Attrition Rate, Employee Count, Salary Distribution, Department-wise Analysis.
Dashboard Highlights  
   - Attrition analysis by department and role  
   - Salary vs attrition insights  
   - Age and experience trends  
   - Interactive filters for dynamic exploration
